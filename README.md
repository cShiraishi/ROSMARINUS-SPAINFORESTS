# ROSMARINUS-SPAINFORESTS
**A Phytochemical Essential Oil Repository of Salvia rosmarinus**

## Overview
ROSMARINUS-SPAINFORESTS is a comprehensive tracking and visualization platform designed to catalog the phytochemical profiles and geographical distribution of essential oils extracted from *Salvia rosmarinus* (Rosemary) populations across Spain. The project integrates climatic and soil environmental factors with high-throughput molecular analysis to elucidate the diversity of natural volatile compounds found within endemic Spanish forests.

## Key Features

### 1. Geographical & Climate Mapping 📍
The repository documents the specific sampling locations encompassing 8 distinct forests across 6 unique regions/provinces in the Iberian Peninsula (Such as Madrid, Navarra, Albacete, Almería, Granada, and Guadalajara). 
It correlates the occurrences with critical environmental stress factors and historical climatic adaptations, including:
* Detailed Relief and Altitude ranges (260 - 1120 m)
* Historical annual temperature bounds (10.9 to 18.0 °C)
* Rainfall variations (267 - 688 mm)
* Soil reaction and pH profiles (Acidic to Alkaline)

### 2. Molecular & Chemical Profiling 🧪
The repository serves as an extensive chemical library, cataloging over 50 isolated volatile organic molecules derived from the multivariate analysis of these rosemary populations. 
For each chemical entity, the database provides:
* Standardized molecular nomenclature
* Retention Time (Rt) methodologies
* Computational **SMILES** (Simplified Molecular-Input Line-Entry System) strings to allow predictive chemoinformatics
* 2D visual molecular graphical representations dynamically generated

### 3. Integrated Raw Data Pipeline 🗃️
For transparency, researchers can query the unadulterated raw data matrices directly from the repository's interfaces, linking the raw field sampling variables (Geographic Data) directly with the laboratory isolation outputs (Chemical Data).

## Objective
The ultimate goal of this repository is to provide ecologists, chemists, and researchers with an open, analytical viewport to understand the climatic resilience, chemotypic variance, and essential oil value of *Salvia rosmarinus* across dynamic Spanish ecosystems.
